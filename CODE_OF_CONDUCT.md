# Code of Conduct

## Our Standards

We expect all participants to:
- Be respectful and constructive.
- Focus critiques on ideas, code, and behavior, not people.
- Assume good intent and ask clarifying questions before escalating.
- Help keep discussions technically rigorous and welcoming.

Unacceptable behavior includes:
- Harassment, discrimination, or intimidation.
- Personal attacks, insults, or hostile trolling.
- Doxxing or sharing private information without consent.
- Sustained disruptive behavior in issues, PRs, or discussions.

## Scope

This code of conduct applies to all project spaces, including:
- GitHub issues
- Pull requests
- Discussions
- Any project-managed communication channels

## Reporting

If you experience or witness a violation, report it privately to repository maintainers.
Use GitHub private channels where possible (for example, direct maintainer contact).

Please include:
- What happened
- Where and when it happened
- Any relevant links/screenshots

## Enforcement

Maintainers may take actions including:
- Warning
- Comment/thread moderation
- Temporary or permanent ban from project spaces

Enforcement decisions are made to protect the project and community.
